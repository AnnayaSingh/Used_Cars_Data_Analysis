<h2>📌 Project Overview</h2>

<p>
This project focuses on performing <b>Exploratory Data Analysis (EDA)</b> 
and data preprocessing on a Used Cars dataset sourced from Kaggle.
</p>

<p>
The objective is to clean, transform, and analyze the dataset to extract 
meaningful insights related to car pricing, fuel types, mileage, engine 
specifications, and other important features.
</p>

---

<h2>🎯 Objectives</h2>

<ul>
<li>Import and explore the dataset</li>
<li>Perform data cleaning and handle missing values</li>
<li>Conduct data reduction</li>
<li>Create new features (Feature Engineering)</li>
<li>Generate statistical summaries</li>
<li>Perform Univariate Exploratory Data Analysis (EDA)</li>
<li>Apply data transformation techniques</li>
<li>Extract insights from the dataset</li>
</ul>

<h2>📊 Dataset Information</h2>

The dataset contains information about used cars including:

- Car Name  
- Year of Manufacture  
- Selling Price  
- Fuel Type  
- Transmission Type  
- Mileage  
- Engine Capacity  
- Ownership Details  

**Dataset Source:** Kaggle

<h2>⚙️ Project Workflow</h2>

<h3>1️⃣ Data Loading</h3>

- Imported necessary libraries  
- Loaded CSV file using Pandas  

<h3>2️⃣ Data Cleaning & Wrangling</h3>

- Handled missing values  
- Removed duplicate records  
- Corrected inconsistent data formats  

<h3>3️⃣ Data Reduction</h3>

- Dropped unnecessary columns  
- Focused on relevant features  

<h3>4️⃣ Feature Engineering</h3>

- Created new derived features  
- Modified existing columns for better analysis  

<h3>5️⃣ Statistical Summary</h3>

- Generated descriptive statistics  
- Analyzed mean, median, standard deviation  

<h3>6️⃣ Exploratory Data Analysis (EDA)</h3>

- Performed univariate analysis  
- Visualized distributions using graphs  
- Identified patterns and trends  

<h3>7️⃣ Data Transformation</h3>

- Applied scaling and formatting techniques  
- Prepared dataset for further analysis  

---

<h2>🔍 Key Insights</h2>

- Certain fuel types show higher resale value.  
- Car prices decrease significantly with age.  
- Mileage and engine capacity impact pricing trends.  
- Transmission type influences market demand.
