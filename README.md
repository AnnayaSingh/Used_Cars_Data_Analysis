##  Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) and data preprocessing on a Used Cars dataset sourced from Kaggle.  

The objective of this project is to clean, transform, and analyze the dataset to extract meaningful insights related to car pricing, fuel types, mileage, engine specifications, and other important features.

---

##  Objectives

- Import and explore the dataset
- Perform data cleaning and handle missing values
- Conduct data reduction
- Create new features (Feature Engineering)
- Generate statistical summaries
- Perform Univariate Exploratory Data Analysis (EDA)
- Apply data transformation techniques
- Extract insights from the dataset

---

## Technologies & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Information

The dataset contains information about used cars including:

- Car Name
- Year of Manufacture
- Selling Price
- Fuel Type
- Transmission Type
- Mileage
- Engine Capacity
- Ownership Details

Dataset Source: Kaggle

##  Project Workflow

### 1️⃣ Data Loading
- Imported necessary libraries
- Loaded CSV file using Pandas

### 2️⃣ Data Cleaning & Wrangling
- Handled missing values
- Removed duplicate records
- Corrected inconsistent data formats

### 3️⃣ Data Reduction
- Dropped unnecessary columns
- Focused on relevant features

### 4️⃣ Feature Engineering
- Created new derived features
- Modified existing columns for better analysis

### 5️⃣ Statistical Summary
- Generated descriptive statistics
- Analyzed mean, median, standard deviation

### 6️⃣ Exploratory Data Analysis (EDA)
- Performed univariate analysis
- Visualized distributions using graphs
- Identified patterns and trends

### 7️⃣ Data Transformation
- Applied scaling and formatting techniques
- Prepared dataset for further analysis

---

## Key Insights

- Certain fuel types show higher resale value.
- Car prices decrease significantly with age.
- Mileage and engine capacity impact pricing trends.
- Transmission type influences market demand.
